## Git & GitHub

**What is Git?**<br>
&nbsp; &nbsp; &nbsp; &nbsp; Git is a Version Control System (VCS) use to:- <br>
&nbsp; &nbsp; &nbsp; &nbsp; 1. Track changes in code.<br>
&nbsp; &nbsp; &nbsp; &nbsp; 2. Save Different Version of Projects.<br>
&nbsp; &nbsp; &nbsp; &nbsp; 3. Work safely without losing code.<br>
&nbsp; &nbsp; &nbsp; &nbsp; 4. Collaborate with other Developers.<br>

**In Simple Words:-**<br>
&nbsp; &nbsp; &nbsp; Git helps us to manage and track code our code history. 

**Interview:-**<br>
&nbsp; &nbsp; &nbsp; Git is a Version Control System (VCS) that track changes in source code.<br>
