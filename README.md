## üìå About React JS

React JS is a **JavaScript library** developed by **Facebook (Meta)** for building **fast, scalable, and interactive user interfaces**, especially **Single Page Applications (SPA)**.

React follows a **component-based architecture**, making code reusable, readable, and easy to maintain.
